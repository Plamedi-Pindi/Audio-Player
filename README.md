# Audio Player

Mini reprodutor de audio 

## Funcionalidades 🔗

  * Reproduzir audio
  * Trocar as faixas de músicas para traz ou frente

  ### Sobre os audios
  O sistema possui 5 faixas de músicas de seguintes autores:
    1. Nair Nani
    1. Mercy 
    1. Sandra Mbuyi

## Ferramentas 🛠️

* React.js
* Typescript
* Tailwindcss
